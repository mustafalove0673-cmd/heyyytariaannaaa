# ✦ A R I A N A — Yapay Zeka Asistanı

**Ariana**, nefes alan, gözleriyle seni takip eden ve seninle konuşabilen interaktif bir robot asistanıdır.

## Özellikler

- 🕐 Gerçek zamanlı saat ve takvim
- 🤖 Animasyonlu robot yüzü (nefes alma, göz takibi)
- 🎤 Ses tanıma ve sentez (Web Speech API)
- 🔮 7/24 sürekli dinleme modu
- ✨ Partikül animasyonları
- 🎨 Özel Art Deco / Cyber-elegance tasarım

## Teknolojiler

- HTML5 / CSS3 (OKLCH renk uzayı, CSS animasyonları)
- Vanilla JavaScript (Canvas partiküller, Web Speech API)
- SVG (Robot yüzü)
- Google Fonts (Cinzel, Jost, Space Grotesk)
